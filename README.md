# Deezer App Clone (Test)


## Installation

Use the package manager npm or yarn to install

```bash
yarn
```
or

```bash
npm install
```
## Running

```bash
yarn dev
```
## Testing
```bash
yarn test
```

## Storybook
```bash
yarn storybook
```
